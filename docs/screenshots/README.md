# Screenshots

Tambahkan screenshot di folder ini untuk ditampilkan di README utama.

Rekomendasi screenshot yang menarik untuk portofolio:

1. `homepage.png` — Tampilan homepage dengan banner, kategori, flash sale, dan testimonial
2. `catalog.png` — Halaman katalog dengan filter dan sorting
3. `product-detail.png` — Halaman detail produk
4. `seller-dashboard.png` — Dashboard seller
5. `seller-products.png` — Halaman kelola produk seller
6. `checkout.png` — Halaman checkout dengan pilihan alamat
7. `admin-categories.png` — Halaman admin kelola kategori
8. `mobile-bottom-nav.png` — Tampilan mobile dengan bottom navigation

Tips:
- Gunakan resolusi 1280x720 atau 1920x1080
- Ambil screenshot di browser dengan mode responsive untuk desktop dan mobile
- Pastikan data dummy terlihat realistis
- Gunakan browser tanpa extension yang mengganggu tampilan
